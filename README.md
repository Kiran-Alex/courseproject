# CourseCo

This project is Still in Development

 # Technologies Used:

React,
Chakra UI,
Recoil,
Axios,
NodeJS,
StripeAPI,
TypeScript,
MongoDB,
AWS

**Features:-**

**Users**

1. Signup and Sign in
2. Purchase Courses with Stripe API
3. View Purchased Courses

**Admin**

1. Signup and Sign in
2. Create Courses
3. Edit Courses
